Les liens

Les syntaxes des liens

Lien absolu (externe)
Exemple: <a href="adresse du site"> texte lien </a>

=> ceff.ch

Lien sur le disque dur
Exemple: <a href="adresse du site"> texte lien </a>

=> ceff.ch

Infobulle
Exemple: <a href="adresse du site" title="Infobulle"> texte lien </a>

=> ceff.ch

Cible du lien _blank
Exemple: <a href="adresse du site" target="_blank"</a>(Le lien s'ouvre dans un nouvel onglet)

=> ceff.ch

Cible du lien _self
Exemple: <a href="adresse du site" target="_self"</a>(Le lien s'ouvre dans la même fenêtre)

=> ceff.ch

Le lien envoie de courrier simple
Exemple: <a href="mailto:nom@domain.ch">contact</a>

=> contact

Le lien envoie de courrier(spécifier le sujet de l'email)
Exemple: <a href="mailto:nom@domain.ch?subject=Cours HTML">contact</a>

=> contact

Le lien envoie de courrier(spécifier le destinataire)
Exemple: <a href="mailto:nom@domain.ch?subject=Cours HTML&cc=info@ceff.ch">contact</a>

=> contact
